# Claude Code Rules

These extend `core.md` with Claude Code-specific guidance. Used in `~/.claude/CLAUDE.md` (global) or `./CLAUDE.md` (per-project).

## Additional Rules

- **No large inline HTML in JS files.** If HTML is more than a few lines, put it in a separate `.html` file.
- **Document as you go.** When making changes, update README.md and CLAUDE.md with relevant project info (new routes, architecture changes, env vars, etc.).
